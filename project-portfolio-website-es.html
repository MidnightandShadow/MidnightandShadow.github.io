Sitio Web - Portafolio Digital

Sumario del Proyecto

Uno de los líderes del Club de Desarrollo de Video Juegos de Northeastern University me inspiró a crear mi sitio web. Durante su presentación sobre los diferentes métodos para conocer varios profesionales, el demostró las ventajas de crear un portafolio digital para profesiones de Desarrollos de Video Juegos y/o Programadores. Desde ese momento, yo aprendí los conceptos fundamentales de HTML y CSS a través de ‘LinkedIn Learning’ y así programé este mismo sitio web.

Este proyecto es mi primera incursión dentro de ‘front-end web dev’ y he tenido que consultar otras páginas como ‘CSS-Tricks’ Me esforcé por usar las técnicas correctas que aprendí para garantizar que el sitio web sea navegable, de fácil actualización, y receptivo, aunque los tamaños de pantallas estándar cambien a futuro.

Responsabilidad(es)

  • Programé todo el código HTML y CSS
  • Diseñé la estética del sitio web
  • Elegí los tipos de letra y otros graficas
  • Reuní, organicé, y ajusté las imágenes
  • Escribí el contenido en inglés y español
  • Abordé las preocupaciones de accesibilidad

Lo Que Aprendí

Este proceso me demostró, directamente, cuanto trabajo toma crear un sitio web moderno desde cero. He desarrollado una nueva visión a través de la cual ahora percibo la red. Debido a la gran cantidad de herramientas para construir un sitio web sin programación previa, tan popular estos días, yo no había tomado en cuenta lo que constituía el ‘front-end web dev’; pero ahora, tengo una nueva apreciación por esta especialización. También he sentido que libertador es poder personalizar cada elemento individualmente en mi sitio web.

Los SVGs de los botones de las redes sociales fueron descargados de ‘Font Awesome’ y usado dentro del terma de la Licencia Atribución/Reconocimiento 4.0 Internacional.
